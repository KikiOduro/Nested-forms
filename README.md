# Employee Management Form

A multi-tab "Add Employee Details" form built with React and Reactstrap, designed to capture complete employee onboarding information across five sections:

- **Personal** — name, contact info, nationality, and profile picture upload
- **Organizational** — staff ID, department, position, and salary grade details
- **Payment Info** — dynamic table for adding multiple payment options via modal
- **Other Info** — payroll hours and payslip notes
- **General Ledger** — GL account mapping for salary, tax, and contribution fields

Built with reusable components, controlled form state, and conditional field logic (e.g. fields that unlock once a related dropdown is selected).

## Tech Stack
- React
- Reactstrap (Bootstrap components for React)
- React Icons
- Vite
